Number Guessing Game

The Way it Works

  • Enter a maximum range.
  • A random number would be generated within 1 to max range.
  • You have to guess the number
BEWARE You have a limited number of lives. Guess Wisely!!
HINT The message "though very close" implies that you are in +-5 range of the hidden number!!

This is a Number Guessing Game completely made with JavaScript,hence you will have to work in the console log. Press 'F12' to access the browser's Inspect Window. Have Fun!!
